### <i class="fa fa-question-circle"></i> Overview

`seaPiper` is an interactive viewer for [sea-snap](https://github.com/bihealth/seasnap-pipeline) and other 
bulk transcriptomic analysis results. 
It does **not** run analyses; it lets you inspect and compare results that are already computed.

Use the left sidebar to switch between modules:

**Note:** modules are shown only when their required inputs are present in `seaPiperData`.

### <i class="fa fa-list"></i> Modules And Data Requirements

| Module | Sidebar icon | Description |
|---|---|---|
| Gene browser | <i class="fa fa-dna"></i> | Upper panel: Browse through gene expression analysis results; lower panel: show gene expression for selected gene |
| Volcano plots | <i class="fa fa-mountain"></i> | Plot log~2~ fold change vs p-values |
| Disco plots | <i class="fa fa-chart-line"></i> | Comparison of different contrasts |
| Tmod browser | <i class="fa fa-project-diagram"></i> | Gene set enrichment analysis results |
| Panel plots | <i class="fa fa-grip-vertical"></i> | Summary of gene set enrichment analysis results for multiple contrasts |
| PCA | <i class="fa fa-cube"></i> | Principal component analysis |
| Workflow Info | <i class="fa fa-info-circle"></i> | Meta-data, covariates, model used etc. |
| Help | <i class="fa fa-question-circle"></i> | This page |

### <i class="fa fa-random"></i> Cross-Module Navigation

- Selecting genes in **Volcano** or **Disco** switches the view to **Gene Browser plot**, *unless* the 
  selected gene is already selected in the **Gene browser**.

- Selecting a gene set in the **Panel plot** switches the view to **Tmod browser**.

### <i class="fa fa-lightbulb"></i> Practical Notes

- Some tabs show a **Dataset** selector only when multiple datasets are loaded.
- Download buttons save publication-ready PDF figures from module-specific plots.
- For reproducibility details (R/package versions), use **Workflow Info -> Session info**.
