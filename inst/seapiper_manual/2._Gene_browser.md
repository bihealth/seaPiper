### <i class="fa fa-dna"></i> Gene Browser

The Gene browser is split into two boxes:

- **Gene table** (top): browse differential expression results.
- **Gene info** (bottom): inspect one selected gene in detail.

### <i class="fa fa-table"></i> Gene Table

Left controls:

- **Contrast** selector.
- **Filter** toggle.
- **Filter by abs(LFC)**.
- **Filter by FDR**.
- **Direction** (`Any`, `Up`, `Down`).

Right side:

- Results table (one row per gene for the selected contrast).
- Use the `â–¶` button in a row to select a gene for the lower panel.

### <i class="fa fa-chart-area"></i> Gene Info

Left controls:

- Plot display controls (covariate-driven aesthetics, depending on available metadata).
- **PDF** download for the current plot.

Main area:

- **Plot** tab: expression profile of the selected gene.
- **Contrast overview** tab: selected gene summarized across contrasts.

### <i class="fa fa-link"></i> Integration With Other Tabs

- Selecting genes in **Volcano plots** or **Disco plots** can automatically open this tab.
- Annotation columns may include link-outs (for example to Ensembl/NCBI) when available.

> If this tab is missing, at least one of `cntr`, `annot`, `covar`, or `rld` is not available.
