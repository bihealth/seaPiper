### <i class="fa fa-project-diagram"></i> Tmod Browser

Use this tab to inspect gene-set enrichment (`tmod`) results.

Layout:

- **Gene set enrichment overview** (top)
- **Evidence plot** (bottom)

### <i class="fa fa-filter"></i> Gene Set Enrichment Overview

Left controls include:

- **Contrast** selector.
- Gene-set **database** selector.
- **Sorting** selector (when available).
- **Filter results** toggle.
- Thresholds: **AUC**, **FDR**, **Min. gene set size**, **Max. gene set size**.

Main pane includes:

- **Results** table of enriched sets.
- **Upset plot** tab (set-overlap summary) with options for plot type and size.

Use the `â–¶` button in the results table to choose a gene set for the evidence plot.

### <i class="fa fa-chart-line"></i> Evidence Plot

- **Save plot** button.
- Module info table.
- Tabs:
  - **Plot**: enrichment evidence plot.
  - **Genes**: gene table for the selected module.

### <i class="fa fa-random"></i> Integration

Selecting a gene set in this tab (or in **Panel plot**) updates the active `gs_id` and can bring focus back to **Tmod browser**.

> If this tab is missing, at least one of `tmod_res`, `tmod_dbs`, `tmod_map`, or `tmod_gl` is not available.
