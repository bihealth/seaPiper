### <i class="fa fa-chart-line"></i> Disco Plots (Discordance / Concordance)

Disco plots compare two contrasts gene-by-gene.

- **X-axis**: log2 fold change in contrast 1
- **Y-axis**: log2 fold change in contrast 2

Each point is one gene.

### <i class="fa fa-sliders-h"></i> Controls

- **Contrast 1** / **Contrast 2** selectors.
- Matching mode (`matchby`, when available).
- **Automatic scale** toggle.
- Manual **Min** / **Max** axis sliders.
- **Show top labels** and **Top labels (N)**.
- Manual gene labels input (`comma-separated IDs`) + update button.
- **Save plot to PDF**.

### <i class="fa fa-mouse-pointer"></i> Point Selection

- Hover to identify genes.
- Click a point to select one gene.
- Click-and-drag (brush) to select multiple genes.

Selected genes are listed in the right panel; using the selection button opens detailed gene views in **Gene browser**.

### <i class="fa fa-info-circle"></i> Interpretation

Disco coloring reflects concordance/discordance scoring (Domaszewska et al., 2017):

- Concordant genes: similar direction in both contrasts.
- Discordant genes: opposite directions between contrasts.

> This plot needs at least two contrasts. If fewer are available, the UI shows an informational message.

Reference:

Domaszewska, T., et al. *Scientific Reports* 7.1 (2017): 1-13.
