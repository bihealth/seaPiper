### <i class="fa fa-mountain"></i> Volcano Plots

Volcano plots summarize differential expression for each contrast.

- **X-axis**: log2 fold change
- **Y-axis**: significance scale derived from p-values

### <i class="fa fa-sliders-h"></i> Controls

- **Dataset** selector (`All datasets` is available when multiple datasets are loaded).
- **P-value threshold**.
- **Log2 FC threshold**.
- **Same X scale** / **Same Y scale** across facets.
- Figure size and font size controls.
- **Save plot to PDF**.

### <i class="fa fa-mouse-pointer"></i> Interactions

- Hover for gene details.
- Click or brush to select genes.
- Selected genes are listed in the right panel with action buttons.

Using a selected gene can update `gene_id` and open **Gene browser** for expression-level inspection.

> If this tab is missing, `cntr` data are not available.
