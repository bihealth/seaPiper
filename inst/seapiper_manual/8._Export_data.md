### <i class="fa fa-download"></i> Export Data

This tab uses the `bioshmods::fileExport*` module to export objects currently loaded in `seaPiperData`.

### <i class="fa fa-file"></i> What Can Be Exported

Depending on availability, the export list includes:

- Differential expression contrasts (`cntr`)
- Gene annotation (`annot`)
- Covariates (`covar`)
- Expression matrix (`rld`)
- PCA coordinates (`pca`)
- Raw tmod objects (`tmod_res`, `tmod_dbs`, `tmod_map`, `tmod_gl`)
- Pipeline configuration (`config`)
- Full `seaPiperData` object

### <i class="fa fa-file"></i> Export Formats

- **xlsx** for tabular structures (single table, list of tables, or nested table lists)
- **zip** for "save all" from nested table lists (one xlsx per selection)
- **rds** for generic/raw objects

### <i class="fa fa-lightbulb"></i> Tips

- Use tabular exports for spreadsheet workflows.
- Use RDS exports to preserve full object fidelity for R-based downstream analyses.
- Filenames are automatically sanitized for safe download names.
