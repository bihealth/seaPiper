### <i class="fa fa-info-circle"></i> Workflow Info

This tab summarizes pipeline metadata and sample annotation tables.

### <i class="fa fa-database"></i> Dataset Selection

- If multiple datasets are loaded, a **Select data set** drop-down appears.
- All overview tables are updated for the selected dataset.

### <i class="fa fa-clipboard-list"></i> Panels

- **Overview**
  - Organism and taxon
  - Design formula
  - Number of contrasts
  - tmod databases
  - Filtering thresholds used in preprocessing

- **Contrasts**
  - Contrast IDs and titles

- **Covariates**
  - Sample-level metadata table
  - Export buttons: Copy / CSV / Excel

- **Session info**
  - Current R session and package versions
  - Useful for reproducibility/debugging

> If this tab is missing, at least one of `config` or `covar` is not available.
