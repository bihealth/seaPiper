### <i class="fa fa-cube"></i> Principal Component Analysis (PCA)

This tab visualizes sample-level PCA computed from normalized expression.

### <i class="fa fa-sliders-h"></i> Controls

- **Dataset** selector (when multiple datasets are available).
- Aesthetic mappings for sample metadata:
  - Color grouping
  - Symbol grouping
- **3D** toggle.
- Axis component selectors:
  - X component
  - Y component
  - Z component (used in 3D mode)

### <i class="fa fa-chart-scatter"></i> Plot

- Interactive Plotly PCA panel (zoom, pan, rotate in 3D).
- Useful for checking sample clustering and covariate-driven separation.

> If this tab is missing, at least one of `pca` or `covar` is not available.
